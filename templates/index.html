BEM VINDO AO SISTEMA DE GERENCIAMENTO DE INFORMAÇÃO


Objetivo

O Sistema de Gerenciamento de Informação tem como objetivo melhorar a eficiência da comunicação entre o setor de Recursos Humanos (RH) e o setor de Tecnologia da Informação (TI) da nossa empresa. Com ele, pretendemos simplificar o gerenciamento de informações dos nossos funcionários, fornecendo uma maneira rápida e precisa de acessar e atualizar esses dados.

Como usar ?

Utilizar o sistema é simples e intuitivo. Aqui estão as etapas básicas para começar:

  • Adicionar Funcionário
    • Essa função adiciona um funcionário ao banco de dados.
    • Todos os campos são obrigatórios.
    • Editar Funcionário
    • Essa função edita os dados de um funcionário que já está no banco de dados.
    • Depois de atualizar, os dados serão sobescritos perdendo a informação anterior.
    • Pode ser utilizada para corrigir erros de digitação ou excluir um item.

  • Lista de Funcionários
    • Essa função exibe todos os dados de funcionários que estão no banco de dados.
    • É possível pesquisar na barra de texto que se encontra acima das lista.
    • É possível editar e excluir nas opções ao lado de cada funcionário.
    • A lista se mantém sempre atualizada de forma automática.
